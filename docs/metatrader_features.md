# MetaTrader 5 Feature Parity

This document outlines the features of MetaTrader 5 that will be implemented in the web application.

## Advanced Charting

*   [ ] Real-time charts with multiple timeframes (1M, 5M, 15M, 30M, 1H, 4H, 1D, 1W, 1M)
*   [ ] Candlestick, bar, and line chart types
*   [ ] A comprehensive library of technical indicators (e.g., Moving Averages, RSI, MACD, Bollinger Bands)
*   [ ] Drawing tools (e.g., trend lines, Fibonacci retracements, support and resistance levels)

## Trading Functions

*   [ ] Placing market and pending orders (buy limit, sell limit, buy stop, sell stop)
*   [ ] Setting stop-loss and take-profit levels for orders
*   [ ] Modifying and canceling pending orders
*   [ ] Closing open positions

## Account Management

*   [ ] Viewing account balance, equity, and margin
*   [ ] Displaying a detailed history of trades
*   [ ] Generating account statements

## News Feed

*   [ ] Real-time financial news from various sources
*   [ ] Filtering news by symbol or keyword

## Economic Calendar

*   [ ] A calendar of important economic events
*   [ ] Filtering events by country and importance
